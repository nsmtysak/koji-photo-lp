# 工事写真台帳アプリ — 拡販LP（広告ビジュアル版）

現場のiPhoneだけで工事写真帳PDFを作成・送付できる「工事写真台帳アプリ」の、
**会社単位での導入問い合わせ**を獲得するためのランディングページ（単体・1ページ）。

- 読み手: 現場監督・職人（使う人）
- ゴール: 「会社での導入を相談する」問い合わせ
- 方向性: アプリ風ミニマルではなく、**現場の実写を主役にした鮮やかな広告ビジュアル**

## ファイル構成

```
LP/
├── index.html        # LP本体（CSS/JS同梱・1ファイル完結）
├── images/           # Pexelsの実写素材（商用利用可）
│   ├── foreman-b.jpg # ヒーロー背景：現場で連携する2人
│   ├── foreman-c.jpg # 最終CTA背景：陽光の職人 ／ PDF例
│   ├── doc-a.jpg     # 悩みセクション：図面を見る作業者
│   ├── doc-b.jpg     # 撮影指示：図面を確認する元請け×協力会社
│   ├── site-a/b.jpg  # PDF例サムネ／鉄骨
│   ├── site-d.jpg    # 引用バンド背景：鉄骨
│   └── foreman-a/e.jpg # 予備カット
├── README.md
└── .claude/launch.json  # ローカルプレビュー用（python http.server）
```

## 設置

`index.html` と `images/` フォルダをそのままサーバー／GitHub Pages 等に置くだけ。ビルド不要。

## 公開前に差し替える箇所（重要）

`index.html` 内の `▼▼ … ▼▼` コメントが目印です。

| 箇所 | 現在の値 | 差し替える内容 |
|---|---|---|
| 問い合わせメール | `mailto:your-company@example.com` | 実際の問い合わせ先メールアドレス |
| 電話番号 | `tel:000-0000-0000` / 表示 `000-0000-0000` | 実際の電話番号・受付時間 |
| 提供会社名（フッター） | `〔提供会社名〕` | 実際の会社名 |
| アプリURL（設定済み） | `https://nsmtysak.github.io/koji-photo-app/` | 公開URLが変わる場合は更新 |

> メール問い合わせはアプリ本体と同思想で `mailto:`（件名・本文の雛形入り）。
> フォーム運用にする場合はCTAの `href` をフォームURLに差し替えてください。

## デザイン方針

- **ヒーロー**：現場写真を全面に敷き、深いネイビーのグラデーションで文字を可読化。アンバー(#ffb400)でキーワードを強調する広告ヘッドライン。
- **配色**：ブランドの青(#0a84ff)＝信頼／アンバー＝活力・CTA／ネイビー(#0d1626)＝引き締め。
- **構成**：ヒーロー → 数字バンド → 悩み(写真分割) → 3ステップ(ダーク) → 引用バンド(全面写真) → 特長6 → Before/After → 撮影指示(写真) → FAQ → 最終CTA(写真) → フッター。
- スクロールリベール／レスポンシブ／`prefers-reduced-motion` 対応済み。

### タイポグラフィ・記号の統一（重要）

- 書体は **Noto Sans JP**（Google Fonts, 400/500/700/900）に一本化。端末(iPhone/Android)で別書体になる問題と、日本語と英数字が別フォントで描画される不統一を解消。
- 絵文字・特殊記号（📧 ✉ ▶ ✓ ● × など）は**すべてインラインSVGアイコン**に置換。書体に依存せず、どの端末でも同じ見え方になる。
- ブランドマークは、山＋太陽＋紙飛行機の装飾的アイコンを廃し、**フラットで硬質な「写真入り書類」マーク**へ刷新。

### スマホでの改行最適化

- 見出しは**文節を `<span class="nb">`（`white-space:nowrap`）で束ねる**ことで、狭い画面でも語の途中で折れない。
- PC専用の強制改行は `<br class="pc">`（640px以下で非表示）。スマホでは文節単位の自然改行に切り替わる。
- 検証済み：iPhone幅(375px)で横スクロールゼロ、全見出しが文節できれいに2〜4行に収束。

## 写真の出典・ライセンス

実写素材はすべて [Pexels](https://www.pexels.com/) から取得。Pexelsライセンス（商用利用可・帰属表示不要）。
別の写真に差し替える場合は `images/` 内のファイルを同名で置き換えるか、`index.html` のパスを変更してください。

## ローカルプレビュー

`.claude/launch.json` に簡易サーバー設定済み。`python -m http.server 4173` 等で確認できます（本番配信では不要）。
